UMBRELLO

(A., n.d.) Umbrello is an open-source unified modelling tool and code generator ideal for the analysis and design phase in software development. Umbrello supports multiple UML diagrams such as case diagrams, sequences, classes, components, collaboration, deployment and a list of entities, activities and states; however, in the case of modelling, the XSS Cross-site scripting attack, I found that the sequence diagram would be the diagram to use to display the detailed flow of the XSS attack process, however when showing the architecture of the software development process.

Sequence Diagram

The sequence diagram displays a time sequence depicting the processes and objects in a message exchange between other objects and processes. Regarding the XSS attack displayed in Figure 1, the sequence diagram would be one UML tool for showing the elements or parts of a web-based application and software processes as they interact over time and are organised according to “objects (horizontally) and time (vertically)” (Visual Paradigm, n.d.) this would allow for a detailed overview of the software development process for the proposed software.

Visual Paradigm

“Visual Paradigm is a software design tool designed for agile software projects.” (Visual Paradigm, n.d.) And has a built-in agile development methodology in the software, which helps build systems in an agile way.
